What's New

4.01.00.00

Note:The minimum version of Code Composer Studio required is CCSv11 and compiler version is 21.6.0.LTS

  • F28003x Launchpad support
  • SYSCFG - Added support for following peripherals
    • CMPSS
    • MEMCFG
    • CLA
    • PMBus
    • ePWM/HRPWM
    • HRCAP
  • SYSCFG - Added board view support
  • DCSM Tool for F28003x
  • Provided APIs for configuring LFU registers on F28003x
  • New examples
    • SD FATFS Library Example - Added FatFS update and exFAT support, SPI SD card demo
    • ADC example for open/short detect feature
    • DCC Continuous clock monitoring
  • Flash API for F28003x updated with functional safety features

Revision History

4.00.00.00

    • Added support for F28003x device – Driverlib, Bitfield , SysConfig and Libraries updated for F28003x
    • CLB Tool updated for Linking TILE design with CLB Hardware resource
    • Migration of projects to C2000 compiler version v21.6 LTS.
    • Software Diagnostic Library (SDL) v3.00.00 with support for F2838x device
    • Added Training folder containing the lab exercises
    • Added support for PMBus over I2C feature
    • CLAmath : Optimized 256, 512, 1024 FFT assembly routines
    • ACI Motor Control Benchmark: CLA support for F28004x and F2837x demonstrating two execution modes – CLA executing entire control service routine, C28 offloading part of compute to CLA
    • COFF flash API libraries for F2838x C28x and F28002x.
    • Flash Kernels : F2838x, F28002x, F28003x examples
    • Flash usage examples : F28002x ROM Flash API build configuration for Flash API usage example, ECC Test Mode example
    • C2000 Crypto Lib : The AES library under \library\security has now been renamed to crypto as now it includes SHA algorithm support as well.There is a compatibility break in the AES library as the API names have changed. This has been documented in the C2000_Crypto_Library_User_Guide.pdf in the path libraries\security\crypto\c28
    • Ethernet NDK based example (tcpEchoF2838X) has been descoped from this release as NDK is no longer supported.
